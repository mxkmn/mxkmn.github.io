Chronus is connected to the Tracer service, which provides statistics on usage and crashes of the application for 90 days. This means that the server will receive information about each application launch to count the number of audience, and also error and device information when the application crashes.

The information transmitted to the Tracer service includes the installation id, information about the device (brand, model, Android version), about the application (from where it was installed, its version), as well as some other information to which I do not have access and which is required only for the correct functioning of Tracer (the full list can be found in the privacy policy of the service).

A list of the data to be received is listed below:

a) Device information: device type and model, processor architecture, system language, OS version, screen resolution, amount of available memory, application version, Internet Service Provider, date and time.

b) Information about technical identifiers that identify the application: Android ID.

The calendar access permission is granted only for the application functioning, the calendar data is not transferred anywhere.


К chronus подключён сервис Tracer, который предоставляет статистику использования и падений приложения в течение 90 дней. Это обозначает то, что на сервер будет передаваться информация о каждом запуске приложения для подсчёта количества аудитории, а также информация об ошибке и устройстве при возникновении падении приложения.

Среди информации, передаваемой сервису Tracer, имеется id установки, присутствует информация об устройстве (бренд, модель, версия Android), о приложении (откуда установлено, версия), а также некоторая другая информация, к которой у меня нет доступа и которая требуется исключительно для корректной работы Tracer (с полным списком можно ознакомиться в политике конфиденциальности сервиса, пункт "О Конечных пользователях").

Список получаемых данных указан ниже:

а) Информация об устройстве: тип и модель устройства, архитектура процессора, язык системы, версия ОС, разрешение экрана, количество доступной памяти, версия приложения, Интернет-провайдер, дата и время.

б) Информация о технических идентификаторах, которые идентифицируют приложение: идентификатор Android ID (в устройствах Android).
